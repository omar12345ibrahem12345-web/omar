Your 5-Minute Quick Start

Pick ONE path below. Do it now. Come back for the rest.

Path 1: Just Chat

Easiest entry point. Try a frontier model conversation.

  1. Go to chatgpt.com or claude.ai
  2. Create free account
  3. Paste this prompt: "Act as a helpful assistant. I'm new to AI. Explain [topic I care about] simply, then give me 3 practical ways to use AI for it."
  4. Hit enter. Read. Follow up.
Detailed guide →

Path 2: AI-Powered Search

Replace Google for complex questions. Get synthesized answers with citations.

  1. Go to perplexity.ai (no account needed)
  2. Ask: "What are the latest developments in [your field]?"
  3. Click sources. Verify. Go deeper.
Detailed guide →

Path 3: Code Assistant

If you write any code, this is the highest-ROI starting point.

  1. Install Codeium (free) or GitHub Copilot in VS Code
  2. Open a project. Type a comment: // function to parse CSV and return array of objects
  3. Press Tab. Review. Edit. Learn.
Detailed guide →

Path 4: Local LLMs (Private)

Run models offline. No data leaves your machine. Free forever.

  1. Install Ollama (Mac/Windows/Linux)
  2. Run: ollama run llama3.2
  3. Or try LM Studio for GUI + model browsing
Detailed guide →

Prompting Fundamentals

The skill that compounds. Better prompts = exponentially better results.

Core Structure: RACE

A memorable framework for every prompt:

  • Role — "Act as a senior marketing strategist"
  • Action — "Write a launch email sequence"
  • Context — "Product: AI course for developers. Audience: 2-5 yrs exp"
  • Example/Expectation — "Tone: concise, practical. Format: 3 emails, <200 words each"
# Before (weak)
"Write a marketing email for my AI course."

# After (RACE)
Role: Senior B2B marketing strategist for developer tools
Action: Write 3-email launch sequence for "AI for Developers" course
Context: Target: 2-5 yr exp devs. Pain: overwhelmed by AI hype, want practical skills. Course: 4 weeks, hands-on, $299.
Expectation: Tone: peer-to-peer, no fluff. Each email <200 words. Include one code snippet in email 2.

Advanced Techniques

  • Few-shot: Include 2-3 input/output examples
  • Chain-of-thought: "Think step by step" or "Reason through this"
  • Self-consistency: Run 3x, take majority vote (for factual Qs)
  • Delimiters: Use ### INPUT ### / ### OUTPUT ###
  • Negative constraints: "Do NOT use marketing jargon. No exclamation marks."
  • Persona + Format: "Output as JSON matching this schema: {...}"

Common Pitfalls

  • 🤷 Vague: "Make it better" → ✅ Specific: "Shorten by 30%, active voice, add metrics"
  • 📝 Overloading: 5 tasks in one prompt → ✅ Decompose: Chain prompts
  • 🎭 No role: Generic output → ✅ "Act as [expert] with 15 yrs exp in [field]"
  • 🔒 No constraints: Hallucination risk → ✅ "Only use provided docs. Say 'unknown' if not found"
  • 🔄 Single shot: "It didn't work" → ✅ Iterate: "Good, now make it more concise"

System Prompts (Reusable)

Save these as custom instructions / system prompts:

📝 General Assistant 
You are a helpful, concise, and knowledgeable assistant.
- Be direct. No fluff, hedging, or unnecessary apologies.
- If unsure, say so. Don't hallucinate.
- Use formatting (bullets, code blocks, tables) for readability.
- Ask clarifying questions when ambiguous.
- Prioritize actionable, practical advice.
💻 Code Reviewer 
You are a senior software engineer doing code review.
- Focus on: correctness, security, performance, maintainability
- Be specific: file:line, suggest exact changes
- Praise good patterns. Explain WHY for suggestions.
- Consider context: legacy code, team conventions, constraints.
- Output: Summary → Critical Issues → Suggestions → Nitpicks

Structured Learning Path

From beginner to practitioner. Free resources prioritized.

1

Week 1: Foundations & Hands-On

  • Complete quick start paths above (2-3 hours total)
  • Course: AI for Everyone (Andrew Ng, Coursera) — free audit
  • Read: "A Survey of Large Language Models" (skim)
  • Build: Project 1 (RAG) or Project 6 (Custom GPT)
2

Week 2: Prompting & Evaluation

  • Guide: Prompt Engineering Guide (DAIR.AI)
  • Course: ChatGPT Prompt Engineering (DeepLearning.AI) — free
  • Practice: Awesome ChatGPT Prompts — copy, modify, test
  • Build: Project 2 (Content Pipeline) or Project 4 (Data Analysis)
3

Week 3: Technical Depth

  • Course: Generative AI with LLMs (DeepLearning.AI)
  • Read: "Language Models are Few-Shot Learners" (GPT-3 paper)
  • Hands-on: Hugging Face Transformers tutorials
  • Build: Project 3 (Code Migration) or local LLM setup
4

Week 4: Specialization & Portfolio

  • Pick a track: RAG/Agents, Fine-tuning, MLOps, AI Product, Research
  • Course: DeepLearning.AI Short Courses (many free, 1-2 hrs each)
  • Build a portfolio project. Document on GitHub + blog post.
  • Join communities: HF Discord, r/LocalLLaMA, X AI Community
