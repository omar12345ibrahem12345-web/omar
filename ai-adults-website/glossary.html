AI Glossary

Key terms and definitions for navigating the world of Artificial Intelligence.

AGI (Artificial General Intelligence)
AI with human-level capability across all cognitive tasks. Not yet achieved.
Alignment
Ensuring AI systems pursue intended goals and values, not just specified objectives.
Attention Mechanism
Neural network component allowing models to weigh input relevance dynamically. Core of Transformers.
Backpropagation
Algorithm for computing gradients in neural networks, enabling weight updates via chain rule.
Bias (in ML)
Systematic error from flawed assumptions. Also: societal bias reflected in training data.
Chain-of-Thought
Prompting technique eliciting step-by-step reasoning from LLMs.
Compute (FLOPs)
Floating-point operations. Standard measure of training/inference computational cost.
Deep Learning
Neural networks with many layers. Learns hierarchical representations automatically.
Diffusion Model
Generative model learning to reverse noise addition process (Stable Diffusion, DALL-E).
Embedding
Dense vector representation where semantic similarity = geometric proximity.
Emergent Abilities
Capabilities appearing abruptly at scale, not present in smaller models.
Fine-tuning
Adapting pre-trained model to specific task/domain with additional training.
Foundation Model
Large model trained on broad data, adaptable to many downstream tasks (BERT, GPT, CLIP).
Generative AI
AI creating new content (text, images, code, audio) vs. analyzing/classifying.
GPU (Graphics Processing Unit)
Parallel processor essential for deep learning training/inference (NVIDIA H100, A100).
Hallucination
Confident but incorrect/ungrounded output. Fundamental to probabilistic generation.
Inference
Running trained model on new inputs to get predictions. Deployment phase.
LLM (Large Language Model)
Transformer model trained on massive text corpora for next-token prediction (GPT, Llama, Claude).
Loss Function
Mathematical objective minimized during training (cross-entropy, MSE).
MoE (Mixture of Experts)
Architecture routing inputs to specialized sub-networks. Enables scaling parameters efficiently (Mixtral).
Multimodal
Model processing multiple data types: text, image, audio, video (GPT-4o, Gemini).
Overfitting
Model memorizes training data, fails on new data. Combatted by regularization, more data.
Parameters (Weights)
Learned numerical values defining model behavior. "7B" = 7 billion parameters.
Prompt Engineering
Crafting inputs to elicit desired outputs. Includes few-shot, chain-of-thought, RAG.
RAG (Retrieval-Augmented Generation)
Combining retrieval (search) with generation. Grounds LLM in external knowledge.
RLHF (Reinforcement Learning from Human Feedback)
Aligning models via human preference data. Key to ChatGPT, Claude behavior.
Scaling Laws
Empirical power laws: performance improves predictably with compute, data, parameters.
Supervised Learning
Training on labeled (input, output) pairs. Classification, regression.
Temperature
Sampling hyperparameter controlling randomness. 0=deterministic, 1=distribution, >1=creative.
Token
Basic unit of text processing (subword). ~0.75 words/token in English.
Transformer
Architecture using self-attention. Foundation of all modern LLMs (Vaswani et al. 2017).
Training Run
Single end-to-end training process. Can cost $10K-$100M+ for frontier models.
Zero-shot / Few-shot
Task performance with 0 or N examples in prompt, without weight updates.
